# 组织架构 Specification

## Overview
组织架构提供公司部门和同事的可视化视图。用户可以通过双栏布局浏览部门树和同事列表，查看实时在线状态，使用智能搜索快速找到特定同事，并直接发起聊天或查看详情。

## User Flows
- 在左侧部门树中浏览或搜索部门
- 右侧显示选中部门的同事列表
- 使用顶部搜索框快速搜索姓名、部门、职位
- 查看同事在线状态（在线、离开、离线）
- 点击同事卡片发起聊天或查看详细信息

## UI Requirements
- 双栏布局：左侧部门树（280px），右侧同事列表（flex-1）
- 顶部搜索框：支持姓名、部门、职位、拼音搜索
- 部门树：可展开/折叠的层级结构，显示部门名称和人数
- 同事列表：头像、姓名、职位、部门、在线状态、快速操作按钮
- 在线状态指示器：绿色圆点（在线）、黄色圆点（离开）、灰色圆点（离线）
- 快速操作：发起聊天、查看详情
