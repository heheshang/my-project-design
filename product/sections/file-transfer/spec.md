# 文件传输 Specification

## Overview
文件传输功能支持局域网内用户之间的文件传输。用户可以通过拖拽或文件选择器发送文件，在统一列表中管理所有传输（发送和接收），查看详细的传输进度信息，支持暂停、继续、取消操作。传输历史记录所有传输，包括失败的传输。

## User Flows
- 拖拽文件到聊天窗口或联系人头像进行发送
- 通过文件选择器选择文件发送
- 在传输列表中查看所有正在进行的传输
- 暂停、继续、取消正在传输的文件
- 接收文件时选择保存位置
- 查看传输历史记录，了解所有传输状态

## UI Requirements
- 统一传输列表，显示发送和接收的文件
- 每个传输项显示：文件名、发送者/接收者、文件大小、传输进度条、传输速度、剩余时间
- 传输状态指示：等待中、传输中、已暂停、已完成、已取消、失败
- 支持的操作：暂停、继续、取消传输
- 传输历史视图，按时间倒序排列
- 拖拽发送视觉反馈
